GarageApp | Formulario

En esta vista, se muestra un formulario que sirve tanto para insertar como modificar un vehículo.

En él, hay múltiples campos, algunos obligatorios para rellenar con el objetivo de insertar o modificar:

  • Imagen (No obligatorio): Por defecto, mostrará 'Sin imagen'. Para poder agregar una imagen, basta con darle al botón ubicado en la derecha, y aceptando los permisos (si lo desea) se mostrará su galería, y de ahí, podrá seleccionar una imagen.
  • Marca (Obligatorio): En este campo se debe de introducir la marca del vehículo. No debe de estar vacío y no debe se sobrepasar los 30 carácteres.
  • Modelo (Obligatorio): En este campo se debe de introducir el modelo del vehículo. No debe de estar vacío y no debe se sobrepasar los 30 carácteres.
  • Año: (Obligatorio): En este campo se debe de introducir el año del vehículo. No debe de estar vacío y debe de cumplir que, el año esté en un rango en específico, entre 1903 y el año siguiente al actual. Ejemplo, si estamos en 2020, el año máximo permitido es 2021.
  • Tracción (Obligatorio): En este campo se debe de introducir el tipo de tracción del vehículo. No debe de estar vacío y no debe se sobrepasar los 30 carácteres.
  • Combustible (Obligatorio): En este campo se debe de seleccionar el tipo de combustible del vehículo. Por defecto, hay un tipo seleccionado, puedes elegir entre las diferentes opciones que hay disponibles. Si en el caso de que el tipo de combustible del vehículo que desea insertar no está entre las opciones, en la parte derecha del campo, hay un botón (+) que le brinda la posibilidad de agregar manualmente un tipo de combustible.
  • Fecha de matriculación (Obligatorio): En este campo se debe de introducir la fecha de matriculación. Dicha fecha debe de cumplir un formato en concreto: DD/MM/YYYY. Si no desea escribir la fecha manualmente, existe un botón en el lado derecho que le mostrará un calendario para poder seleccionar la fecha más cómodamente.
  • Edición especial (No obligatorio): Este campo brinda la opción de seleccionar si el vehículo a insertar es una edición especial o no.

Si algunos de los campos no cumple con las condiciones, el formulario se presentará de la siguiente manera:

Cuando el formulario cumpla con las condiciones, se podrá insertar el vehículo al darle al botón de 'Guardar'. Si se ha insertado correctamente, se devolverá a la vista del listado y mostrará un mensaje de información.

En el caso de que se esté modificando un vehículo, al darle dicho botón de 'Guardar', hará el mismo protocolo pero mostrando un mensaje diferente.

Para poder eliminar el vehíclo, si estamos modificando, aparecerá un nuevo botón al lado de 'Guardar', llamado 'Borrar'. Al darle a dicho botón, mostrará un mensaje de confirmación para borrar o no el vehículo. Una vez se ha confirmado, se borrará el vehículo, se devolverá al listado y mostrará un mensaje de información.
